# ML-Core-Classification-Regression-Study

## 🔢 Handwritten Digits Classification
* **Objective:** Multi-class classification of 8x8 pixel images.
* **Key Insight:** Applied Support Vector Machines (SVM) and evaluated performance using a confusion matrix to identify digit-specific bias.
* **Visuals:** Included feature intensity maps to demonstrate how pixel weights affect classification.

## 🚗 Auto MPG Fuel Efficiency Prediction
* **Objective:** Regression analysis to predict vehicle fuel consumption based on technical specifications.
* **Data Processing:** Handled non-linear relationships between "Horsepower" and "MPG".
* **Evaluation:** Optimized the model to achieve a low RMSE, ensuring reliable fuel economy estimation.
